## Installation

follow this step to clone and running the project :
### Clone this project

```bash
git clone https://github.com/SenaPutra01/shayna-backend
```
Open this project using code editor

### Running Project
```bash
composer install
```
```bash
php artisan key:generate
```
```bash
php artisan migrate
```
```bash
php artisan serve
```

Go to http://localhost:8000/
